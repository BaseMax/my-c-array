# my-c-array

## 🗂️ Suggested Directory Layout

```
my-c-array/
├── Makefile
├── build.sh
├── build.bat
├── libmyarray.a  (after build)
├── src/
│   ├── array_append/
│   │   ├── array_append.c
│   │   └── array_append.h
│   └── ...
```

## Build

Linux:

```
chmod +x build.sh
bash build.sh
```

or

```
make
```

Windows:

```
build.bat
```
